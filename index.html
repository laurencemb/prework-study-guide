Cascading Style Sheets

  • A margin indicates the space around the outside of an element.
  • Padding indicates the space around the outside of the content inside an element.
  • There are 3 kinds of CSS: Inline, internal css and external css
  • Inline is written within the HTML is quick instant and permanant but must all be updated individually if you want to make changes
  • internal Css the css code is written directly in to the head of a document and allows for classes and ID's making it more flexible
  • External CSS is written on a seperate .css sheet and a link is placed in the head of the html, this makes it easier to access and update and keeps code with different functions seperate.
  • * is a univeral css selector, everything under this selector will beaffectedby the rules/declarations
  • selector defines the element or attributed to be targeted
  • Declarations contain the property to be applied and the value eg: colour:red
